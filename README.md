# 控制工程讲义板书

基于 HTML/CSS/JavaScript 的网页版控制工程多媒体课件，采用黑板风格的沉浸式展示界面，适合课堂教学与自学。

## 内容涵盖

教材: *Modern Control Systems, 14th Ed.* (R. C. Dorf, R. H. Bishop)

- 第 1 章 — 控制系统导论 (Introduction to Control Systems)
- 第 2 章 — 系统的数学模型 (Mathematical Models of Systems)
- 第 4 章 — 反馈控制系统特性 (Feedback Control System Characteristics)
- 第 5 章 — 反馈控制系统的性能 (The Performance of Feedback Control Systems)
- 第 6 章 — 线性反馈系统的稳定性 (The Stability of Linear Feedback Systems)
- 第 8 章 — 频域响应方法 (Frequency Response Methods)
- 第 9 章 — 频域稳定性 (Stability in the Frequency Domain)

## 使用说明

直接打开 `controlEng1to9.html` 即可使用。课件为单页应用，无需后端服务器。

### 基本操作

| 操作 | 方式 |
|------|------|
| 翻到下一页 | 点击 **下一页** 按钮 / 按 `→` 方向键 |
| 翻到上一页 | 点击 **上一页** 按钮 / 按 `←` 方向键 |
| 逐步显示内容 | 点击页面空白区域 / 按 `Space` 空格键 |
| 整页显示 | 勾选 **整页显示** 复选框，再翻页即可一次性显示全部内容 |

### 讲课词与朗读

| 操作 | 方式 |
|------|------|
| 显示/隐藏讲课词 | 点击 **讲课词** 按钮 / 按 `N` 键 |
| 语音朗读讲课词 | 勾选 **读** 复选框，再点击 **讲课词** / 按 `N` 键 |
| 调节语速 | 拖动 **语速** 滑块 |
| 切换语音 | 从语音下拉列表中选择不同的合成语音 |

### 导航

| 操作 | 方式 |
|------|------|
| 打开/关闭目录 | 点击 **目录** 按钮 / 按 `T` 键 |
| 跳转到指定页面 | 在跳转输入框中输入页码后按 Enter，或点击 **Go** 按钮 / 按 `G` 键快速聚焦跳转框 |

### 画笔工具

| 操作 | 方式 |
|------|------|
| 启用/关闭画笔 | 点击 **画笔** 按钮 / 按 `P` 键 |
| 切换颜色 | 点击黄色、红色或绿色色块按钮 |
| 调整粗细 | 拖动粗细滑块 |
| 清除笔迹 | 点击 **清除** 按钮 |

### 其他特性

- **KaTeX 公式渲染**：页面中的数学公式使用 KaTeX 引擎实时渲染
- **响应式布局**：页面自适应窗口大小，支持滚动浏览
- **目录快捷跳转**：目录面板按章节组织，点击可快速跳转至对应幻灯片
- **板书风格界面**：模拟黑板质感的深色背景，适合课堂投影展示
